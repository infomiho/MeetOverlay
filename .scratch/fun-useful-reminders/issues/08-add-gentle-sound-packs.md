# Add gentle sound packs

Type: AFK
Status: Done

## What to build

Let users choose a short bundled reminder sound so the app feels more personal without becoming annoying. The default should stay tasteful, and the selected sound should be used by the alerting path.

## Acceptance criteria

- [x] Settings include a sound selection control with a small set of bundled sounds.
- [x] The current sound remains the default option.
- [x] The user can preview each sound from Settings.
- [x] The selected sound persists across launches.
- [x] Reminder playback uses the selected sound.
- [x] Tests cover preference persistence and selected-sound resolution without requiring audio playback.
